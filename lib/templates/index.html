## Hi, I'm HTML Decks
*Press the right arrow key on your keyboard or swipe to learn more about me.* --- I'm a [free and open source way](https://github.com/vzvenyach/htmldecks) to host professional slide decks. --- I'm built using [Reveal.JS](http://lab.hakim.se/reveal-js/) and inspired by [Biggie](https://github.com/tmcw/biggie). --- To use me, write slides in [markdown](http://daringfireball.net/projects/markdown/syntax) and separate each slide with three dashes. --- Like This! ![](/lib/img/slide_img.png) --- It's that simple to create web presentations, like the one you're looking at right now. --- If you want a robust tool, consider [Slides](https://slides.com). --- But if you want to hack up a presentation real quick, register for free here: ---
